# Interactive Iron Man Mask

Arduino-based wearable mask with moving servo and LED lighting effects.

## Components
- Arduino
- Servo motor
- RGB LEDs
- Push buttons

## Features
- Servo opens and closes the mask
- Buttons control LED color
- Debounce logic for stable button input

## Technologies
- Arduino (C++)
- Embedded programming
